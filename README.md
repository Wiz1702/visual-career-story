# Visual Career Story

Single-page, animated portfolio for Wisdom Akanwe featuring hero profile, skills, experience, education, projects, involvement, and contact links (email/LinkedIn/GitHub).

## Preview

Visit the live site (or run locally) to explore accomplishments, featured projects, and ways to connect.

## Getting Started

```sh
npm install
npm run dev
```

## Build & Preview

```sh
npm run build
npm run preview
```

## Tech Stack

- Vite, React, TypeScript, React Router
- Tailwind CSS, shadcn/ui
- framer-motion for animations
- @tanstack/react-query, lucide-react icons
